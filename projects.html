  • PoisedPMS

    • PoisedPMS is a project management system written in Java, used to store, track and update relevant information about projects, its customers, architects and contractors for a small structural engineering company called Poised. This program utilizes Java OOP, debugging, refactoring and documentation principles to enable the program to fulfill the following:

    - Read details about exisiting projects from a text file;
    - Capture details about new projects;
    - Update the due date of existing projects;
    - Update the contractor's contact number or email address of an existing project;
    - Update the project fee paid to date;
    - View completed, incomplete or overdue projects;
    - Finalize existing projects and write finalized projects to a text file.

  • PoisedPMSRemastered

    • PoisedPMSRemastered is an enhanced version of PoisedPMS. The functionality of the system remains similar with the exception that this program interacts with a database. PoisedPMSRemastered is a project management software system that aims to capture, track and update information relevant to various building projects and the respective individuals (e.g. customers, structural engineers) involved in each of Poised's (small structural engineering company) projects. The program interacts with an external database called PoisePMS used for the capturing, tracking and storing all relevant project details. The system must able to achieve the following:

    - Read details about existing projects and people from the PoisePMS database;
    - Capture information about new projects and individuals involved in the projects and write these details to the database;
    - Update the due date of incomplete proojects and write these to the database;
    - Update and write amended contact details of respective contractors to the database;
    - Update the project fee paid to date and write the updated amount to the PoisePMS database;
    - Enable users to view incomplete, overdue and completed projects from the database;
    - Finalize existing projects once all outstanding project fees have been paid in full and generate invoices to customers for outstanding balances;
